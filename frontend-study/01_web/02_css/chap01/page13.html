2.자손선택자와 후손 선택자

자손: 바로 하위요소들/ 후손: 하위요소들 전부

        * 자손선택자
        a>b{
            스타일속성:값;
        }
        * 후손선택자
        a b{
            스타일속성:값;
        }

3.동위선택자

동위관계(같은레벨) 에서 뒤에 위치한 태그를 선택할때 사용한다.

 
        * a뒤에있는 b요소 하나만선택
        a+b{
            스타일속성:값;
        }
        * a뒤에있는 모든 b 요소 선택
        a~b{
            스타일속성:값;
        }
